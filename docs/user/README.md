## 🎯 クイックスタート

初めてFalCommandを使用する場合：

1. [インストールガイド](./installation.md)に従ってアプリをインストール
2. [クイックスタート](./quick-start.md)で基本操作を習得
3. [設定ガイド](./configuration.md)で好みに合わせてカスタマイズ

## 💡 主な機能

- **高速検索**: アプリケーション、ファイル、コマンドを瞬時に検索
- **キーボード操作**: マウスを使わずに効率的な操作
- **カスタマイズ**: テーマ、ショートカット、検索対象の設定
- **プラグイン**: 電卓、翻訳、天気予報などの拡張機能
- **クロスプラットフォーム**: Windows、macOS、Linux で利用可能

## 📞 サポート

問題が解決しない場合：

1. [FAQ](./faq.md)で類似の問題を確認
2. [トラブルシューティング](./troubleshooting.md)で問題を診断
3. [GitHub Issues](https://github.com/varubogu/falcommand/issues)で報告

## 🔄 アップデート

- FalCommand は定期的にアップデートされます
- 新機能や改善については [リリースノート](https://github.com/varubogu/falcommand/releases) を確認してください
- 自動更新の設定方法は [設定ガイド](./configuration.md) を参照してください
